计算逻辑与数据来源说明

合计值计算逻辑

CBD核心区合计租金 = Σ(各子市场租金 × 存量) / Σ(各子市场存量)

= (287.87×217.96 + 273.06×146.76 + 247.48×63.57 + 226.80×218.22 + 226.52×111.91 + 169.50×145.59) / 904.02

= 238.78 元/㎡/月

全市合计租金 = (CBD核心区合计 × CBD存量 + 新兴商务区合计 × 新兴存量) / 总存量

= (238.78×904.02 + 158.80×875.39) / 1787.31

= 201.43 元/㎡/月

注：各区租金按对应市场存量加权平均，空置率为各子市场的算术平均。

徐汇区AI产业政策汇总（2025-2026）

两处候选物业均位于徐汇区，可享受以下政策。

企业落户与开办
  • 新注册/迁入AI企业，实缴资本达2000万以上，最高按5%开办奖励，上限2000万元
  • 年度主营收入首次突破2亿/10亿/30亿，分别奖励100万/300万/500万元
  • 总部型企业最高奖励2000万元
研发投入与算力
  • 研发投入占营收5%以上，按30%支持，年度最高5000万元
  • 购买/租用算力，最高按实际投入30%奖励，年度最高2000万元
  • 模速空间入驻企业可"免申即享"100万元等价算力大礼包
大模型专项（徐汇区通用）
  • 高质量数据集：最高奖励300万元
  • AI大模型产品销售：按合同额15%，最高1000万元
  • 采购无关联AI产品：按采购额10%，年度最高300万元
  • 与高校共建研发机构：最高500万元
人才与税收
  • 顶尖人才购房补贴最高500万元 + 子女入学绿色通道
  • 青年人才每月最高2000元租房补贴，最长3年
  • 研发费用加计扣除 + 高新企业优惠，十年综合税负降低约40%
租金与办公成本
通用园区政策
• 部分园区最高1500万元办公用房补贴
• 龙华万科/星扬西岸均可申请区级通用政策

申报建议

1. 政策按照"从高不重复"原则执行，具体需经徐汇区相关部门认定

2. 建议看房时向招商方索取书面政策说明，并确认是否协助对接徐汇区科委

3. 提前准备：营业执照、技术说明、融资证明、大模型备案材料（如有）
