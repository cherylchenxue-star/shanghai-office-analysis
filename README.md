# MX · 上海总部选址与写字楼市场分析

> 交互式单页 HTML 报告，整合上海市写字楼租赁市场数据分析与 MX 上海总部选址对比两大模块。

## 在线预览

https://cherylchenxue-star.github.io/shanghai-office-analysis

## 项目概览

本项目将两份独立的 HTML 报告合并为一个带有顶部标签切换的单文件页面，方便在同一界面中浏览市场宏观数据与具体物业选址信息。

### 标签页一：上海市写字楼租赁市场数据分析
- 上海市各区甲级写字楼租金热力图
- 2020–2025 年租金走势与空置率变化趋势
- 商圈租金分布散点图
- 各区租金/物业费详情折叠面板
- 数据来源：2025 年 Q4 市场监测

### 标签页二：MX · 上海总部选址对比
- 四处候选物业（恒基星扬、龙华万科、西岸智塔、东航滨江）核心指标对比表
- 各物业详细参数卡片（租金、交通、层高、配套、优势与劣势）
- Leaflet 地图标注与通勤路线可视化
- 考察清单与政策要点速查

## 技术栈

- **HTML5** + **Tailwind CSS**（CDN）
- **ECharts** — 数据可视化图表
- **Leaflet** — 地图与路线展示
- **Font Awesome** — 图标
- 原生 JavaScript，无构建工具，单文件即可运行

## 本地查看

直接浏览器打开 `index.html` 即可，无需本地服务器或依赖安装。

## 数据来源

- 写字楼租金数据：上海市商业地产市场公开监测数据（2025 年 Q4）
- 选址数据：2026 年 4 月 22 日实地考察信息
